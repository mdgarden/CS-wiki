# CS Wiki

## "오늘 배운 것" 말고, "오늘 질문한 것"
- 기본적인 1일 잔디는 이슈나 디스커션을 오픈하는 것으로 만드는 것을 목표로 함
- 질문이 구체적으로 형성된 경우에는 이슈, 다소 범위가 커져 정리와 구체화가 필요한 경우에는 디스커션에 오픈 후 세부 사항을 이슈에 추가해나감
- 클로즈 된 이슈들은 Wiki에 정리
- Wiki에 담기는 내용들은 내가 나의 언어로 다른 사람에게 충분히 설명할 수 있을 정도로 내가 이해했다는 것을 뜻함
- 출처 등은 해당 페이지 내에서 서술
